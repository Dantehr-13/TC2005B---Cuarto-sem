¿Cuál es la diferencia entre internet y la World Wide Web?

El internet principalmente está compuesto de infrestructura física, protocolos, sistemas de nombres de dominio y la red mundial (WWW). En pocas palabras, la World Wide Web está contenida dentro de lo que es internet.

¿Cuáles son las partes de una URL?

  • Protocolo
  • Subdominio
  • Dominio
  • Subcarpeta
  • Slug
  • Parámetros

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

  • GET: Solicita un recurso, como un HTML del servidor web
  • HEAD: Solicita solo el encabezado del archivo, sin el mismo
  • POST: Se utiliza para enviar un recurso en específico
  • PUT: Similar a POST, con la diferencia de que este puede ser ejecutado varias veces
  • PATCH: Modificaciones parciales de un recurso específico
  • DELETE: Elimina un recurso específico

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

POST, ya que los datos que se registran se envían en el cuerpo del HTTP.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

GET, ya que es el principal método de obtención de datos y recursos.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

El código de respuesta 200 indica que la solicitud de datos fue exitosa, siendo la respuesta estándar en páginas web.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

El error 404 indica que, aunque el servidor logró establecer conexión exitosamente, el recurso no se ha encontrado. Esto podría suceder por varias razones, en algunas de ellas, como por ejemplo la eliminación de la página o algún enlace roto, se responsabiliza al desarrollador por no encontrar los recursos.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

El código 500 (Internal Server Error) indica un fallo inesperado en el servidor que impide obtener el recurso solicitado por el usuario.

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Siendo un método que rompe con los ciclos de CSS con el fin de imponer el propio, no es una acción tan recomendable, ya que impone su prioridad por sobre los otros estilos del documento.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

En un aspecto de diseño, afecta prinxipalmente la percepción de la página, el como es percibida por la gente. Además, El elegir una imagen poco favorable puede derivar en problemas de legibilidad y rendimiento.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Siendo ambas medidas disponibles para el mismo uso, realmente en situaciones en donde no sea un requisito la exactitud del formato del tamaño del texto, es indiferente. En otro caso, en donde se solicita una medida exacta, la precisión de los pixeles es más confiable.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Mejora el rendimiento precisamente por la disminución del tamaño del archivo, eliminando todos los caracteres innecesarios, espacios en blanco, comentarios entre otras cosas sin alterar su funcionalidad original.

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje compilado y orientado a objetos, mientras que javaScript es un lenguaje interpretado basado en prototipos, funcional principalmente en desarrollo web.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • new Date(): fecha y hora actual.
  • getFullYear(): Obtiene el año.
  • getMonth(): Mes.
  • getDay(): Dia de la semana.
  • getTine(): Timestap en milisegundos.

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • push(): Añade uno o más elementos al final de un arreglo
  • pop(): Elimina el último elemento de un arreglo
  • shift(): Elimina el primer elemento de un arreglo y lo devuelve
  • map(): Crea un nuevo arreglo con los resultados de aplicar una función a cada uno de los elementos.
  • filter(): Crea un nuevo arreglo con todos los elementos que cumplan la condición implementada.

¿Cómo se declara una variable con alcance local dentro de una función?

Se define diréctamente dentro del cuerpo de una función o bloque de código.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Disminuye la legibilidad, dificulta la depuración y reduce la modularidad de código.
