TaskFlow

Aplicación web desarrollada con Node.js puro para el Laboratorio 08 de TC2005B. Permite agregar tareas con prioridad, guardarlas en el servidor y visualizarlas en tiempo real. Sin frameworks, sin base de datos — solo HTTP, archivos y lógica limpia.

Rutas del servidor

GET / Página principal
GET /tareas Ver tareas guardadas
GET /acerca Esta página
POST /agregar Guardar nueva tarea
GET /api/tareas Tareas en texto plano
